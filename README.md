# Hanshika_Portfolio

A responsive personal portfolio website built with HTML, CSS, and JavaScript to present my projects, skills, and achievements. The portfolio includes my resume, GitHub and LinkedIn profiles, and contact details, offering a centralized platform to explore my work and connect with me.
## 🚀 Features

* 👤 Personal profile section with photo
* 📄 Resume download/view option
* 💼 Projects showcase with source code links
* 🏆 Achievements and skills display
* 🔗 Integration with GitHub and LinkedIn
* 📧 Contact section with email details
* 📱 Responsive design for different devices
## 🛠 Technologies Used

* HTML – Structure of the website
* CSS – Styling and layout
* JavaScript – Interactivity and dynamic behavior
